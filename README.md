# Roel Sadiang-abay — Portfolio

Personal portfolio site for Roel Sadiang-abay, BS Information Technology student at ISUFST, showcasing Django apps, IoT projects, and AI experiments.

## Run locally

Single static HTML file — no build step needed.

```bash
# just open it
start index.html        # Windows
open index.html         # macOS

# or serve it
python -m http.server 8000
```

## Deploy

Push to GitHub Pages, Netlify, or Vercel — it's plain HTML/CSS/JS, no build required.
